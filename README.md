# MiniDeck
A simple, modern, and offline-capable HTML media player.

## Features

* Plays audio files (mp3, wav, ogg, aac, m4a, webm)
* Supports local files.
* Offline-capable for use without internet
* Modern design with a dark theme
* Shuffle, repeat, and loop playlist features
* Volume control
* Visualizer with a dot pulse animation
* Responsive design for use on mobile devices

## How to Use

1. Open `miniDeck.html` in a web browser.
2. Drag and drop audio files or click the "Add Files" button to add files.
3. Click the play/pause button to play/pause the current file.
4. Click the next/previous buttons to skip to the next/previous file.
5. Click the settings button to access the settings menu.
6. Toggle the shuffle, repeat, and loop playlist features in the settings menu.
7. Adjust the volume using the volume control.
